# ⚙️ Algorithms

Algorithms are step-by-step procedures for solving computational problems efficiently. While **data structures organize data**, **algorithms determine how that data is processed**.

This section covers the fundamental algorithms used in software engineering, coding interviews, and competitive programming. Topics progress from core concepts like sorting and searching to advanced techniques such as dynamic programming and graph algorithms.

---

# 🎯 Learning Objectives

By completing this section, you will be able to:

- Understand how classical algorithms work.
- Choose the most efficient algorithm for a given problem.
- Analyze time and space complexity.
- Implement algorithms from scratch.
- Recognize common algorithmic paradigms used in interviews.

# 📚 Topics Covered

| No. | Topic | Description |
|:--:|--------|-------------|
| 01 | Sorting | Arrange data efficiently using comparison and non-comparison sorting algorithms. |
| 02 | Searching | Locate elements efficiently in arrays and other data structures. |
| 03 | Recursion | Solve problems by reducing them into smaller instances. |
| 04 | Divide and Conquer | Divide problems into independent subproblems and combine their solutions. |
| 05 | Backtracking | Explore all possible solutions while pruning invalid paths. |
| 06 | Greedy Algorithms | Build optimal solutions using locally optimal choices. |
| 07 | Dynamic Programming | Solve overlapping subproblems by storing previous results. |
| 08 | Graph Algorithms | Traverse graphs, find shortest paths, spanning trees, and connectivity. |
| 09 | String Algorithms | Efficient pattern matching, substring searching, and text processing. |
| 10 | Randomized Algorithms | Use randomness to simplify algorithms or improve expected performance. |
| 11 | Mathematical Algorithms | Number theory, modular arithmetic, combinatorics, and related algorithms. |
