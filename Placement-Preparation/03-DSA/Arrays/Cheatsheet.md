# Cheatsheet

## Key Notes

- Indexing starts from 0 in C++.
- Time to access an element by index is $O(1)$.
- Insertion and deletion in the middle are usually $O(n)$.

## Common Patterns

- Traversal
- Prefix sum
- Sliding window
- Two pointers
- Frequency counting

## Common Mistakes

- Off-by-one errors
- Using uninitialized values
- Forgetting to resize when needed
- Accessing invalid indices
