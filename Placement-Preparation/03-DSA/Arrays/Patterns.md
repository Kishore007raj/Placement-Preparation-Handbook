# Patterns

## 1. Traversal

- Recognition: need to inspect each item
- Use when: counting, finding max/min, or printing values

## 2. Prefix Sum

- Recognition: repeated subarray sum queries
- Use when: range sum is required quickly

## 3. Sliding Window

- Recognition: contiguous subarray problem
- Use when: maximize/minimize over a window

## 4. Two Pointers

- Recognition: sorted array and pair-based problem
- Use when: searching from both ends
