# Binary Search

## 1. Introduction

Binary Search is a fast searching technique used on sorted data. It repeatedly divides the search space in half and removes one half each step.

## 2. Learning Objectives

- Understand when binary search is applicable.
- Learn the iterative and recursive versions.
- Analyze time and space complexity.
- Apply binary search to sorted arrays and search spaces.

## 3. Prerequisites

- Arrays and indexing
- Sorted order
- Basic loops and conditions

## 4. Learning Order

1. Understand the idea of halving the search space.
2. Learn lower-bound and upper-bound variants.
3. Practice on problems like first occurrence and peak element.
4. Move to advanced binary search patterns.
