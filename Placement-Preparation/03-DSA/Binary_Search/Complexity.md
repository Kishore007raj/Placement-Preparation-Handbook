# Complexity Analysis

| Operation              | Best Case | Average Case | Worst Case |      Space |
| ---------------------- | --------: | -----------: | ---------: | ---------: |
| Search in sorted array |    $O(1)$ |   $O(log n)$ | $O(log n)$ |     $O(1)$ |
| Recursive search       |    $O(1)$ |   $O(log n)$ | $O(log n)$ | $O(log n)$ |
| Find first occurrence  |    $O(1)$ |   $O(log n)$ | $O(log n)$ |     $O(1)$ |
| Find last occurrence   |    $O(1)$ |   $O(log n)$ | $O(log n)$ |     $O(1)$ |
