# Patterns

## 1. Two Pointers

- Recognition: palindrome or reverse-based problem
- Use when: comparing characters from both ends

## 2. Sliding Window

- Recognition: longest substring with conditions
- Use when: maintain a valid window while extending

## 3. Hashing / Frequency Map

- Recognition: anagram, repeated character, or occurrence-based problems
- Use when: fast lookups are needed

## 4. Build New String

- Recognition: need to transform or filter characters
- Use when: removing or rearranging characters
