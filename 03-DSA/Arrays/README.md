# Arrays

## 1. Introduction

Arrays are the most fundamental data structure in programming. They store elements in contiguous memory and allow fast indexing.

## 2. Learning Objectives

- Understand array declaration and indexing.
- Learn traversal, insertion, deletion, and searching.
- Practice common array patterns.
- Apply arrays to interview problems.

## 3. Prerequisites

- Basic C++ syntax
- Loops and conditions
- Functions

## 4. Learning Order

1. Learn indexing and traversal.
2. Practice prefix sums and sliding window.
3. Study in-place manipulation.
4. Move to problems like two sum and subarray sums.
