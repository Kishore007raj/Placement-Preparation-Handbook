# Array Patterns

This document lists all patterns commonly used to solve **Array** problems.

Detailed explanations, templates, recognition techniques, complexity analysis, and practice problems are available in the **04-Algorithm_Patterns** folder.

---

# Core Patterns

* Traversal
* Simulation
* Frequency Counting
* Hashing
* Sorting
* Prefix Sum
* Prefix State (Prefix XOR / Prefix Frequency / Prefix HashMap)
* Difference Array
* Kadane's Algorithm
* Matrix Traversal
* Matrix Prefix Sum
* Two Pointers
* Sliding Window
* Binary Search on Arrays

# Pattern Recognition

| If the problem says...                         | Think...                                   |
| ---------------------------------------------- | ------------------------------------------ |
| Traverse every element                         | Traversal                                  |
| Follow the given operations exactly            | Simulation                                 |
| Count occurrences                              | Frequency Counting                         |
| Fast lookup of values                          | Hashing                                    |
| Rearrange elements                             | Sorting                                    |
| Range Sum                                      | Prefix Sum                                 |
| Prefix XOR / Prefix Balance / Prefix Frequency | Prefix State                               |
| Range Updates                                  | Difference Array                           |
| Maximum Subarray                               | Kadane's Algorithm                         |
| Grid or Matrix                                 | Matrix Traversal                           |
| Rectangle Sum Queries                          | Matrix Prefix Sum                          |
| Sorted Array                                   | Two Pointers / Binary Search               |
| Contiguous Subarray                            | Sliding Window                             |
| Search for Minimum or Maximum Answer           | Binary Search on Answer                    |
| Make the best local choice                     | Greedy                                     |
| Overlapping Intervals                          | Merge Intervals                            |
| Next Greater / Smaller Element                 | Monotonic Stack                            |
| Sliding Window Maximum / Minimum               | Monotonic Queue                            |
| Top K Elements                                 | Heap                                       |
| Frequent Range Queries                         | Segment Tree / Fenwick Tree / Sparse Table |
| Large Coordinate Values                        | Coordinate Compression                     |
| Process Events in Order                        | Sweep Line                                 |
| Offline Range Queries                          | MO's Algorithm                             |
