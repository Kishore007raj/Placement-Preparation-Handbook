# Strings

## 1. Introduction

Strings are sequences of characters used to represent text. They are one of the most common interview topics because many problems involve parsing, matching, or transforming text.

## 2. Learning Objectives

- Understand string storage and indexing in C++.
- Learn common operations like traversal, comparison, and substring extraction.
- Practice pattern-based string problems.
- Apply string techniques in interviews.

## 3. Prerequisites

- Basic C++ syntax
- Arrays and loops
- Knowledge of characters and ASCII

## 4. Learning Order

1. Learn indexing and iteration.
2. Practice palindrome and anagram problems.
3. Study sliding window on strings.
4. Move to substring and pattern matching problems.
