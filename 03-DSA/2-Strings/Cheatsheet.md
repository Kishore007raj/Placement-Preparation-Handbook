# Cheatsheet

## Important Notes

- Strings are commonly processed using indices.
- `s.size()` gives length.
- `s.substr(l, n)` extracts a substring.
- `s.find(x)` searches for a substring.

## Common Patterns

- Two pointers
- Sliding window
- Hashing
- Frequency counting
- Palindrome checks

## Common Mistakes

- Off-by-one errors
- Incorrect handling of whitespace
- Ignoring case sensitivity
- Using wrong bounds for substring operations
